AssetsSelectors Integration Guide

Step 1: Include the assets-selectors.js script tag, which adds the PureJSSelectors object to the window object.
<script src="https://experience.adobe.com/solutions/CQ-assets-selectors/static-assets/resources/assets-selectors.js"></script>
Step 2: Call registerAssetsSelectorsAuthService(...) with imsAuthProps on page load.
const imsAuthProps = { imsClientId: IMS_CLIENT_ID_ASSOCIATED_WITH_YOUR_ORG, imsScope: "AdobeID,openid,additional_info.projectedProductContext,read_organizations", redirectUrl: window.location.href, }; // must be called on page load PureJSSelectors.registerAssetsSelectorsAuthService(imsAuthProps);
Step 3: Call renderAssetSelectorWithAuthFlow with AssetSelector props to render.
function App() { const assetSelectorProps = { imsOrg: imsOrg, onClose: () => {} } PureJSSelectors.renderAssetSelectorWithAuthFlow(container, assetSelectorProps); }
